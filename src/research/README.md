# Research

## Interests
My interests in Computer Science include (though not necessary relate to my work so far):
- Models of temporal behaviour:
    - Developed several extensions to the [Clock Constraint Specification Language](https://en.wikipedia.org/wiki/Clock_Constraints_Specification_Language) under the name during my PhD [Modular Real-Time Clock Constraint Specification Language](https://theses.fr/2024COAZ4058)
    - Synchronous languages
- Analysis using [Abstract Interpretation](https://en.wikipedia.org/wiki/Abstract_interpretation)
- Language semantics and compilation
- Formal verification and proof mechanization

## Current work
In the context of HAL4SDV project, I develop Hardware Abstraction Layer (HAL) description for Software-Defined Vehicles, and its analysis.
It includes development of [MRTCCSL language](./software/mrtccsl.md) and its analysis capabilities, [the Domain-Specific Language](https://github.com/jdeantoni/SoftwareDefinedVehicleModelingLanguage) for the vehicle architecture and its translation into the former language.
